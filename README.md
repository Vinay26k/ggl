ggl
===
Google right from your command line
-----------------------------------

Some description

Installation
------------
```sh
gem install ggl
```

Usage
-----
```sh
ggl [query]                          simple google search
ggl -i/--images [query]              google images search
ggl -y/--youtube [query]             youtube search
ggl -s/--stack/--stackoverflow       search stackoverflow (oh yes)

```
